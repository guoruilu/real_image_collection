# 项目约定

本工程下有两个子工程：
- `small_image_collection/` — 给定类别下载 → CLIP 真实性过滤 → YOLO-World 精确**裁剪**目标主体（产物是「物体特写小图」）。
- `scene_image_collection/` — 给定类别下载 → 分辨率过滤 → CLIP 真实性过滤 → YOLO-World 定位目标 + 多检测器扫描共存物体 → **保留整张原图** + JSON 元数据（产物是「带共存物体的场景大图」，用于 EEG 视觉刺激设计）。

两个子工程共用此份约定，日志和文档各自在子工程下维护。

## 约定

1. **每次任务完成要详细记录日志**：在所操作子工程的 `logs/` 目录下追加一条记录，包含日期、任务描述、关键参数、产出位置、遇到的问题与解决方式。
2. **每次任务完成更新所有相关文档**：包括顶层 `README.md`、所操作子工程的 `README.md`、子工程下的 `docs/*.md`、脚本顶部 docstring 等，确保用法示例与当前代码一致。
3. **要求获得真实图片**：不得包括卡通、雕塑、表情包、艺术设计、绘画等。
4. **图片数量不设上限**：最后过滤出来有多少就存多少，不再硬性限制 50 张。
